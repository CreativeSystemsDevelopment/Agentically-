# Agentically-

A curated collection of AI agent frameworks, SDKs, copilot tooling, and agentic coding tools.

## Original Repositories

1. **agent-zero** - Agent Zero AI framework (Python)
2. **copilot-sdk_ml_platform** - Multi-platform SDK for integrating GitHub Copilot Agent into apps and services (TypeScript, MIT License)
3. **agent-framework** - A framework for building, orchestrating and deploying AI agents (Python, MIT License)
4. **awesome-copilot** - Community-contributed instructions, prompts, and configurations (JavaScript, MIT License)
5. **openclaw** - Your own personal AI assistant. Any OS. Any Platform. (TypeScript, MIT License)
6. **claude-agent-sdk-demos** - Claude Code SDK Demos (TypeScript)

## Agentic Coding Tools

7. **open-interpreter** - Natural language code execution assistant (Python) — [openinterpreter/open-interpreter](https://github.com/openinterpreter/open-interpreter)
8. **OpenHands** - Autonomous AI software engineer, formerly OpenDevin (Python) — [All-Hands-AI/OpenHands](https://github.com/All-Hands-AI/OpenHands)
9. **SWE-agent** - Agent that autonomously fixes GitHub issues (Python) — [princeton-nlp/SWE-agent](https://github.com/princeton-nlp/SWE-agent)

## Copilot & AI Assistant Extensions

10. **copilot-docs** - Official GitHub Copilot documentation — [github/copilot-docs](https://github.com/github/copilot-docs)
11. **anthropic-cookbook** - Recipes & patterns for building with Claude — [anthropics/anthropic-cookbook](https://github.com/anthropics/anthropic-cookbook)
12. **claude-code** - Claude's agentic coding tool and SDK (TypeScript) — [anthropics/claude-code](https://github.com/anthropics/claude-code)

## Personal AI Assistants & Agent SDKs

13. **openai-agents-python** - OpenAI's official agent SDK (Python) — [openai/openai-agents-python](https://github.com/openai/openai-agents-python)
14. **pydantic-ai** - Agent framework with type-safe tool calling and structured outputs (Python) — [pydantic/pydantic-ai](https://github.com/pydantic/pydantic-ai)
15. **semantic-kernel** - Microsoft SDK for integrating AI agents into apps (C#/Python/Java) — [microsoft/semantic-kernel](https://github.com/microsoft/semantic-kernel)
16. **llama-agents** - Microservices-based agent orchestration (Python) — [run-llama/llama-agents](https://github.com/run-llama/llama-agents)

Each repository has been cloned in its entirety, preserving full commit history and content.
